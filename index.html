Sound.JS
A Light Weight Class for Playing Audio

Sound.JS is a simple light weight class that will load and play audio files on demand.

Why not just use the default JavaScript Audio object?

Well you can. But using this method requires each instance that begins playing to finish playing before starting another instance.

Try it out below.

Using the Audio object:

No matter how many times you click the sonar pulse button, it will not start a new sound instance until the current one is finished.

The sci-fi typing sound is supposed to repeat for each letter, but as you can hear, it has to wait until each sound instance is completed.

Using: Sound.JS Class

Multiple audio instances are now possible. Notice there's no latency between the audio clips if you click the sonar pulse multiple times.

The sci-fi typing now sounds as it should.
